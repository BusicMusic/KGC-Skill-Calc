⚠ ATTENTION ⚠

There are a few things you should know about these calculators before you use them

  1. The information used on this website is up to date as up the 4th Anniversary update released on October 23rd, 2024
    • All the hero's stats, skills, level passives, and awakenings are based off the information provided in the game as of this date

  2. All the hero's base stats (Health, Attack, Spell Power, etc.) are all calculated as if they're level 20
    • This means the skill damage outputs will be higher if you don't have a lv20 hero, but the difference in skill damage with various relic bonuses will still happen at lower levels, but the numbers won't be as big

  3. All the hero's base stats and skill effects are also calculated as if they're Tier 1 Summoned (one bronze star)
    • This means when a skill says something like "does 90/100/110/120% Damage" that the calculators use the Tier 1 skill version, which would be 90% in this case. This results in heroes of higher tiers doing more damage and having larger numbers in the differences between skill damage when comparing Spell Power to Attack Power relic bonuses

  4. Some of the calculators are pretty straight forward, and almost don't even need to be used to figure out the skill damage, but they're all here for the sake of consistency
    • Some heroes like Aramis have a 1:1 ratio of Skill Damage to Spell Power/Attack with no awakening skill selected, so a calculator isn't necessary to figure out the damage they'll do. Regardless, since their awakening skills can change that damage ratio they're included anyway.

  5. This is a passion project that I'm doing on my own, so some calcs may be wrong
    • If you find something to be wrong, please let me know via Discord (ping "@Stycin Viro" in the official King God Castle discord server) and I'll do my best to fix it ASAP.

  6. There are currently a total of 61 heroes in the game, so there are 31 rows of 2 heroes on the table below
    • To quickly go to a specific hero, use the Find in Page feature (Ctrl+F on Windows or Options→Find in Page on Mobile) and type the hero name, or you can use the button below to instantly go to the bottom of the page instead of scrolling all the way down

Please keep these things in mind as you use the calculator since they can affect the output of your skill damage results
